# LifeMap AI

Компактный прототип сайт-приложения, где:

- пользователь заполняет анкету о своей жизненной ситуации;
- анкета сохраняется в SQLite;
- затем пользователь переходит в AI-чат;
- AI отвечает простым и понятным языком с учетом анкеты и статистики по России;
- в анкете есть блок будущих жизненных рисков.

## Что реализовано

- форма анкеты;
- демо-сценарий входа “через Госуслуги”;
- сохранение профиля и истории сообщений в SQLite;
- AI-чат с двумя режимами:
  - `OpenAI API`, если задан `OPENAI_API_KEY`;
  - `local fallback`, если ключ не задан или запрос к API недоступен;
- статистические ориентиры по России, Москве, Санкт-Петербургу и Краснодару;
- блок будущих жизненных сценариев:
  - совместная жизнь;
  - ребенок;
  - потеря работы;
  - снижение дохода;
  - ипотека или кредит;
  - помощь родителям;
  - здоровье;
  - расставание или развод.

## Дизайн

Визуальная тема переведена на новую палитру:

- `#162660` Royal Blue — заголовки, кнопки, активные элементы;
- `#D0E6FD` Powder Blue — мягкие блоки, фоновые секции, декоративные акценты;
- `#F1E4D1` Bone — теплый базовый фон и крупные поверхности.

## Запуск

```bash
node server.js
```

После этого приложение доступно на [http://localhost:3000](http://localhost:3000).

## Запуск через Docker Compose

```bash
docker compose up --build
```

После этого приложение также будет доступно на [http://localhost:3000](http://localhost:3000).

## Подключение OpenAI API

Создайте `.env` рядом с `server.js`:

```env
OPENAI_API_KEY=ваш_ключ
OPENAI_MODEL=gpt-5.1
PORT=3000
```

Если ключ не задан, сайт все равно работает и отвечает через локальный planner.

## Основные файлы

- `server.js` — сервер, SQLite, API, prompt и fallback-логика;
- `public/index.html` — интерфейс анкеты и чата;
- `public/styles.css` — новый дизайн в Royal Blue / Powder Blue / Bone;
- `public/app.js` — клиентская логика;
- `data/assistant.db` — база данных.

## Возможности чата

Пользователь может спрашивать, например:

- стоит ли переезжать в другой город;
- когда лучше покупать квартиру;
- насколько рискованно решение при будущем ребенке;
- что делать, если доход может снизиться;
- можно ли тянуть ипотеку с учетом семьи и резервов.
